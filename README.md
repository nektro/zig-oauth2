HTTP handler functions to allow you to easily add OAuth2 login support to your Zig application.

## Supported Providers

- Amazon
- Battle.net
- Discord
- Facebook
- GitHub
- Google
- Microsoft
- Reddit
- Gitea
- GitLab
- Mastodon
- Pleroma
- ... and custom ones!
